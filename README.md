# Web-based Gradebook

## Описание проекта
Данный проект представляет собой веб-приложение для автоматизации учебного процесса. Система включает в себя электронный журнал успеваемости, расписание занятий и модуль управления лабораторными работами. Приложение разделено на две основные роли: «Преподаватель» и «Студент».

## Стек технологий
- **Фронтенд:** React, TypeScript, Material-UI
- **Бэкенд:** NestJS, TypeScript
- **База данных:** Microsoft SQL Server
- **ORM:** Prisma

## Требования к окружению
Для успешного запуска проекта на локальном компьютере должны быть установлены:
- Node.js (рекомендуется v18 или выше)
- Microsoft SQL Server

## Инструкция по настройке и запуску

### 1. Настройка базы данных и бэкенда
Откройте первый терминал и перейдите в директорию серверной части приложения. Установите зависимости:

```bash
cd backend
npm install

Создайте файл .env в корне папки backend и укажите строку подключения к вашей базе данных SQL Server:

DATABASE_URL="sqlserver://ЛОКАЛХОСТ_ИЛИ_ИМЯ_СЕРВЕРА;database=Gradebook;integratedSecurity=true;trustServerCertificate=true;"

Сгенерируйте клиент Prisma и синхронизируйте структуру базы данных:

npx prisma generate
npx prisma db push

Запустите сервер в режиме разработки:

npm run start:dev

Бэкенд будет запущен на порту 3000.

2. Запуск фронтенда
Откройте второй терминал, не закрывая первый, и перейдите в директорию клиентской части:

cd frontend
npm install

Запустите клиентское приложение:

npm run dev

3. Управление базой данных (Prisma Studio)
Для удобного просмотра и редактирования записей в базе данных (добавление новых пользователей, предметов, групп) предусмотрен графический интерфейс Prisma Studio. Для его запуска откройте отдельный терминал в папке backend и выполните:

npx prisma studio
